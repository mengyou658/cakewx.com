安装环境

CakeWX 是基于PHP和MySQL技术开发，可同时使用于Windows、Linux平台，环境需求如下：
1、Windows 平台：
Apache2.2 + PHP5.2.8(最低要求) + MySQL5
2、Linux 平台
Apache/nginx + PHP5.2.8(最低要求) + MySQL5 建议使用平台：Linux + Apache2.2 + PHP5.2.8（以上） + MySQL5.0
3、Apache 必须开启mod_rewrite.so模块。
4、PHP必须扩展库和系统函数：
     Curl 组件
     short_open_tag 开启
5、CakePHP version > 2.4.6
6、环境安装完成后，通过浏览器输入地址：http://youwebsit(您自己的网站地址)/ 按照提示进行安装即可。
